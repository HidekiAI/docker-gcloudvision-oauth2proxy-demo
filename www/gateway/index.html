Gateway: SHOULD NOT BE HERE

This is the (nginx-proxy) gateway. You should never land here...

  • If you've landed on this page, you've most likely set your redirect URL incorrectly.
  • This is the "gateway" (via nginx-proxy) page, which should never be seen by the user. It's sole purpose is to redirect the user to the correct page based on either the origin URI (i.e. myhost.mydomain.tld) with or without port and routing paths used.
  • For this excercise, we have two domains: www.mydomain.tld and internal-developers.mydomain.tld (with or without ports). The filenames located in vhosts.d directory indicates the origin (i.e. "internal-developers.mydomain.tld") and its contents indicates its location routing paths.
  • If you've landed on this page, you first want to determine the URI used (i.e. any URI origin that is not www or internal-developers or possibly misspelled origin such as "www.my-domain-with-dash.tld") and just find that string in your source code
